# 📝 To Do List - Projeto CRUD com React + Node.js

Este é um projeto de **To Do List** desenvolvido para praticar os conceitos de uma aplicação **CRUD** completa, utilizando tecnologias modernas tanto no frontend quanto no backend.

---

## 🚀 Tecnologias Utilizadas

### Frontend
- [React](https://reactjs.org/) com `useState` e `useEffect`
- [Axios](https://axios-http.com/) para requisições HTTP
- [React-DatePicker](https://reactdatepicker.com/) para seleção de datas
- [Tailwind CSS](https://tailwindcss.com/) para estilização rápida e responsiva

### Backend
- [Express](https://expressjs.com/) para criação da API
- [uuid](https://www.npmjs.com/package/uuid) para geração de IDs únicos
- [cors](https://www.npmjs.com/package/cors) para permitir requisições cross-origin
- [body-parser](https://www.npmjs.com/package/body-parser) para interpretar o corpo das requisições

---

## 💻 Funcionalidades

- ✅ Criar tarefas
- ✏️ Editar tarefas
- ❌ Deletar tarefas
- 🔍 Buscar por nome
- 📅 Selecionar data prevista com calendário
- 🟢 Visualização do status (Finalizado, Parado, Em andamento)
- ⚠️ Validações visuais para campos obrigatórios
- 🔴 Destaque automático para tarefas com prazo vencido

## 📦 Instalação

### 1. Clonar o repositório

```bash
git clone https://github.com/seuusuario/nome-do-repositorio.git
cd nome-do-repositorio
```

---

## 🖥️ Rodando o Frontend

### Requisitos:
- Node.js instalado

```bash
cd frontend
npm install
npm run dev
```

> A aplicação React iniciará em `http://localhost:3000`

---

## 🔧 Rodando o Backend

### Estrutura do backend:
- Localizado na pasta `backend`
- Cria um servidor local na porta `5000`
- Todas as rotas são baseadas no endpoint `/tasks`

### Para rodar:

```bash
cd backend
npm install
nodemon server.js
```
